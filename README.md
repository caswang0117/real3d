# Tetris

```
cargo run --release
```

## Keymaps

- Left Arrow: Rotate camera left (clockwise top view)
- Right Arrow: Rotate camera right (counter-clockwise top view)
- Down Arrow: Increase block dropping speed
- A: Move tetris left
- D: Move tetris right
- W: Move tetris forward
- S: Move tetris backward
- Enter: Save game
- N: new game
